LE CSS (sous réserve de faire du BootStrap)

J'ai organisé le fichier CSS pour que chacun ait sa partie et dans sa chaque partie, les différentes pages que vous avez à faire.

J'ai déjà créé la balise <main> avec son id dans chaque fichier, ainsi qu'un sélecteur pour chaque page dans le fichier css

Pour manipuler vos balises en css, pensez bien à les faire précéder de l'identifiant qui correspond à la page
ex : si vous mettez un <ul> dans votre code et que vous voulez le stylisé comme suit :

ul {
 color : black;
}

, vous styliserai tous les <ul>, même ceux des autres. Donc pour ça pour utiliser l'identifieur défini précédemment pour 
au moins sélectionner les balises <ul> de votre page 
ex :

#accueil ul {
color : black;
}

idem pour toutes les autres balises (cherchez sélecteur CSS sur google)
